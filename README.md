# Ngcwanguba Megasave - Professional Business Website

A modern, professional website for **Ngcwanguba Megasave**, a comprehensive rural retail store serving communities with quality products and services.

## 🌟 Features

### Website Sections
- **Hero Section**: Welcoming introduction with call-to-action buttons
- **About Us**: Company history, mission, and key statistics
- **Services & Products**: 8 main service categories including:
  - General Groceries
  - Hardware & Tools
  - Bakery Products
  - Fuel Station
  - Beverages & Liquor
  - Agricultural Supplies
  - Health & Wellness
  - Clothing & Textiles
- **Portfolio**: Showcase of store highlights and capabilities
- **Testimonials**: Customer reviews and feedback
- **Pricing Plans**: Three membership tiers (Basic, Loyal Member, Bulk Buyer)
- **Contact Section**: Contact information and contact form
- **Footer**: Navigation links and social media

### Design Features
- **Professional Light & Blue Theme**: Clean, modern color scheme
- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Mobile Navigation**: Hamburger menu for smaller screens
- **Smooth Animations**: Fade-in effects and hover transitions
- **Interactive Elements**: Smooth scrolling, form validation, and scroll animations
- **Modern UI**: Card-based layouts, gradient backgrounds, and shadow effects

## 🚀 Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- A code editor (Visual Studio Code recommended)
- Local server (optional, but recommended)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/zondidalasile-netizen/ngcwanguba-megasave.git
   cd ngcwanguba-megasave
   ```

2. **Open the website**
   - Option A: Double-click `index.html` to open in your browser
   - Option B: Use a local server (recommended)
     ```bash
     python -m http.server 8000
     # Then visit http://localhost:8000
     ```

## 🎨 Customization

### Update Company Information
Edit the following in `index.html`:
- Store name and logo
- Contact details (phone, email, address)
- Services and products list
- Pricing information
- Team members and testimonials

### Change Colors
Edit CSS variables in `style.css`:
```css
:root {
    --primary-blue: #2563eb;
    --secondary-blue: #1e40af;
    --accent-blue: #3b82f6;
    /* ... other colors ... */
}
```

### Add Images
1. Create an `images` folder in the project root
2. Replace placeholder gradients in portfolio section with actual images
3. Update image paths in the HTML
